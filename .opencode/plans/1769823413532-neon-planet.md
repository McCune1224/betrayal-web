## Design Pattern
Replace the soft, bubbly aesthetic with:
- **Sharp corners**: Remove all `rounded-*` classes
- **Visible borders**: Add `border-2` or `border` to define edges
- **Drop shadows**: Add `shadow-*` classes for depth:
  - Large cards: `shadow-xl` or `shadow-2xl`
  - Messages: `shadow-lg`
  - Subtle elements: `shadow-md` or `shadow-inner`
  - Buttons: `shadow` with hover states

## Chat Message Specific Styling

